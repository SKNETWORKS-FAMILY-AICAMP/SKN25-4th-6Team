카드 발급 관련 유의사항
· ALL 우리카드 Infinite의 서비스는 아코르 로열티 프로그램 올 - 아코르 리브
  리미티즈(ALL - Accor Live Limitless) 회원에게만 제공되며, ALL 회원이 아닌
  경우 카드 신청 시 ALL 회원 가입이 필수로 진행됩니다.
· ALL 우리카드 Infinite 신청 시 ALL 회원 신규 가입을 하신 경우, 해당 ALL 회원
  번호는 본인카드 기준으로 채번되며, 이후 가족카드 발급 시 본인 카드의 ALL 회원
  번호로 자동 연결됩니다.
· ALL 우리카드 Infinite 발급 시 입력한 ALL 회원 계정은 발급 초년도 연회비 납부 시
  다음날 자동 연결됩니다.
· ALL 우리카드 Infinite와 연결된 ALL 회원 계정에 아코르 서비스가 적용됩니다.
· ALL 우리카드 Infinite는 1인 2장 이상 중복 소지가 불가능합니다.

1. ALL 리워드 포인트 기프트 제공

14,000 ALL 리워드 포인트 기프트 제공 (연 1회)
· 아코르 공식 홈페이지와 모바일 앱을 통한 호텔 숙박 예약 또는
  호텔 현장 결제 시 ALL 리워드 포인트 사용 가능

※ ALL 리워드 포인트 기프트 제공 기준
· ALL 리워드 포인트 기프트는 발급 초년도 연회비 납부 완료 7일 이내에 본인 ALL 회원 계정에
  14,000 ALL 리워드 포인트가 자동 적립됩니다. (연회비 완납 필수)
  (예시) 김우리 회원, ALL 우리카드 Infinite 5월 15일 발급
  - 6월 25일 : 연회비 청구 후 납부 완료
  - 6월 30일 : 14,000 ALL 리워드 포인트 적립
· 발급 2차년도부터는 전년도 국내외 가맹점 이용실적 1,000만원 이상 이용 시 연회비 납부
  완료 7일 이내에 14,000 ALL 리워드 포인트가 자동 적립되며, 본인 ALL 회원 계정을 통해
  확인 가능합니다. (연회비 완납 필수)
· 전년도 이용실적 산정 기준은 최소 카드발급 월 포함 12개월입니다.
· 가족카드 이용실적은 본인카드 이용실적에 합산됩니다.
· 본인 회원만 제공됩니다. (가족카드는 ALL 리워드 포인트 기프트 제공 불가)
· ALL 리워드 포인트 적립은 ALL 회원 계정이 유효한 경우에 한하여 서비스가 제공됩니다.
· 기프트 제공으로 적립된 ALL 리워드 포인트는 적립 취소가 불가능하며, 전년도 취소매출은
  취소매출 접수년도 이용실적에서 차감합니다.

※ ALL 리워드 포인트 안내
· ALL 리워드 포인트 사용가치 : 1포인트 = €0.02유로(EUR)
· ALL 리워드 포인트는 아코르 공식 홈페이지(ALL.COM) 또는 모바일 앱(ALL.com) 로그인 후
  적립내역 및 포인트 잔액 등 확인 가능합니다.
· ALL 리워드 포인트는 호텔 예약 또는 현장결제 시 현금처럼 사용 가능하며, 대상 호텔 상세
  목록은 아코르 공식 홈페이지(ALL.COM) 또는 모바일 앱(ALL.com)에서 확인 가능합니다.
· 적립되는 ALL 리워드 포인트는 아코르가 운영/관리하는 포인트로, 자세한 사항은 아코르
  공식 홈페이지(ALL.COM) 및 모바일앱(ALL.com) 내 멤버십 약관 등을 참고하시기 바랍니다.

해외 이용 시 청구금액 산출방법
해외 이용 시 청구금액
= (거래미화금액 X 전신환 매도율 ①) + 국제브랜드 수수료 ② + 해외서비스 수수료 ③
