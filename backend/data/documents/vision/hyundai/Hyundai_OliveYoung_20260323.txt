·올리브영 현대카드 리워드란?
올리브영 현대카드 이용에 따라 발행되는 일종의 할인권으로,
올리브영 온·오프라인에서 1천원 이상 결제 시 사용할 수 있습니다.(1매당 1천원 할인)

·올리브영 현대카드 리워드 발행 기준
리워드 적용 금액* 1천원당 1천원권 1매
*올리브영 현대카드 리워드 발행을 위해 대상점별 이용 금액을 적용 비율에 따라
산정한 금액임

| 구분 | 대상점 | 기간 | 리워드 적용 금액 산정 기준 |  |
|------|--------|------|--------------------------|--|
|  |  |  | 적용 비율 | 리워드 적용 금액 한도 |
| 기본 혜택 | 국내외 가맹점 | 상시 | 1% (누적 이용 금액 10만원당 1천원) | 월 3만원 (초과 시 미적용) |
| 추가 혜택 | 올리브영 | 상시 | 3.4% (누적 이용 금액 약 3만원당* 1천원) | 월 20만원 (초과 시 미적용) |
|  |  | 올영세일 | 5% (누적 이용 금액 2만원당 1천원) |  |

*약 3만원 = 29,412원

리워드 적용 금액 산정 및 올리브영 현대카드 리워드 발행 예시
· 국내외 가맹점에서 8만원, 올리브영에서 상시 3만원, 올영세일 기간에 2만 4천원 이용 시 3,020원* 리워드 적용 금액 누적되어 올리브영 현대카드 리워드 3매 발행
* (8만원 x 1% = 800원) + (3만원 x 3.4% = 1,020원) +
(2만 4천원 x 5% = 1,200원) = 3,020원

· 기본 혜택 월 3만원, 추가 혜택 월 20만원 리워드 적용 금액 한도(초과 시 미적용)
· 추가 혜택 대상점은 올리브영(온·오프라인)이미, 기본 혜택 대상점은
그 외 국내외 가맹점임
· 추가 혜택 적용 시 기본 혜택은 중복 적용되지 않음
· 추가 혜택인 올영세일* 기간에는 상시 혜택이 중복 적용되지 않음
*올영세일은 1년에 4번 열리는 올리브영(온·오프라인) 정기 세일로, 정확한 기간은
올리브영 앱 또는 홈페이지(www.oliveyoung.co.kr)에서 확인 가능
· 올영세일 외 기타 세일 기간은 추가 혜택의 '상시' 기간에 해당하여 3.4% 적용

혜택 제공 기준

올리브영 현대카드 리워드 발행 시기
· 매출전표 매입 시점 기준으로 리워드 적용 금액 1천원 누적 시 1~2영업일 후 발행
예시) 20일 올리브영에서 올영세일 기간에 이용한 5만원 매출전표가 21일에 매입된
경우, 22일 리워드 적용 금액으로 누적된 2,500원*에 대해 올리브영 현대카드
리워드 2매 발행. 발행 후 잔여 리워드 적용 금액 500원은 다음 매출전표 매입
시점에 누적되는 리워드 적용 금액과 합산
*5만원 x 5% = 2,500원

올리브영 현대카드 리워드 발행 완료 후 이용 금액 취소 시 반영 기준
· 올리브영 현대카드 리워드 발행 완료된 건의 전체 이용 금액 또는 일부 취소 시, 이후
누적되는 리워드 적용 금액에서 취소 금액에 해당되는 리워드 적용 금액을 제외
예시) 올리브영 현대카드 리워드 발행 완료된 국내외 가맹점 이용 금액 10만원 취소
후, 다음 날 올리브영에서 올영세일 기간에 4만원 이용 시 리워드 적용 금액은
아래와 같음
(4만원 x 5% = 2,000원) - (10만원 x 1% = 1,000원) = 1,000원
· 올리브영 현대카드 리워드 발행 완료된 건의 전체 이용 금액 또는 일부 취소 후
올리브영 현대카드 해지 요청 시, 발행 완료된 올리브영 현대카드 리워드 1매당 1
천원으로 환산하여 현금으로 청구되며, 청구 금액 완납 시 해지 가능

기타 기준
· 리워드 적용 금액은 매출 건별로 소수점 이하 반올림하여 산정(가족 카드 이용에 따른
리워드 적용 금액은 본인 회원에 통합)
예시) 기본 혜택 대상적인 국내외 가맹점에서 올리브영 현대카드로 6,557원 결제 시
66원 리워드 적용 금액으로 누적(6,557원 x 1% = 65.57의 소수점 이하 반올림)
